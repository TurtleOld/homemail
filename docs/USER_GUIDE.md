## Начало работы

### Вход в систему

1. Откройте страницу входа
2. Введите ваш email и пароль
3. Нажмите кнопку "Войти"

После успешного входа вы будете перенаправлены в почтовый ящик.

### Первый запуск

При первом входе вы увидите:
- Пустой список писем (если почтовый ящик пуст)
- Боковое меню с папками
- Кнопку "Написать" для создания нового письма

## Основные функции

### Просмотр писем

- **Список писем** отображается в центральной области
- Используйте **горячие клавиши** для навигации:
  - `J` - следующее письмо
  - `K` - предыдущее письмо
  - `Enter` или `O` - открыть письмо
- **Клик по письму** открывает его содержимое
- **Двойной клик** открывает письмо в новом режиме

### Составление письма

1. Нажмите кнопку **"Написать"** в боковом меню или используйте горячую клавишу `Ctrl+K` (или `Cmd+K` на Mac)
2. Заполните поля:
   - **Кому** - адрес получателя (обязательно)
   - **Тема** - тема письма
   - **Копия** / **Скрытая копия** - дополнительные получатели
3. Напишите текст письма в редакторе
4. Используйте панель инструментов для форматирования:
   - **Жирный**, *Курсив*, <u>Подчеркнутый</u>
   - Списки (маркированные и нумерованные)
   - Цитаты и код
5. Нажмите **"Отправить"**

### Ответ и пересылка

- **Ответить** (`R`) - ответить отправителю
- **Ответить всем** - ответить всем получателям
- **Переслать** (`F`) - переслать письмо

### Управление письмами

- **Выбор писем**: клик по чекбоксу или `Shift+Click` для множественного выбора
- **Перемещение**: перетащите письмо в нужную папку
- **Удаление**: `Delete` или кнопка "Удалить"
- **Пометить прочитанным/непрочитанным**
- **Добавить в избранное** (звездочка)
- **Отметить как важное**

## Настройки

### Подпись письма

1. Перейдите в **Настройки** → **Подпись письма**
2. Введите текст основной подписи
3. Для создания подписей для разных ситуаций:
   - Нажмите **"Добавить подпись"**
   - Укажите название, контекст (рабочая, личная, для автоответа) и содержимое
   - Установите подпись по умолчанию при необходимости

### Тема оформления

1. Перейдите в **Настройки** → **Тема**
2. Выберите **Светлая** или **Темная** тема
3. Изменения применяются сразу

### Настройки интерфейса

В разделе **Интерфейс** вы можете настроить:

- **Плотность отображения**:
  - Компактный - больше писем на экране
  - Обычный - сбалансированный вид
  - Просторный - больше пространства

- **Количество писем на странице** (10-100)

- **Сортировка**:
  - По дате, отправителю, теме или размеру
  - По возрастанию или убыванию

- **Группировка**:
  - Без группировки
  - По дате (Сегодня, Вчера, На этой неделе, по месяцам)
  - По отправителю

### Расширенные настройки

#### Автоответ

1. Перейдите в **Настройки** → **Автоответ**
2. Включите автоответ
3. Укажите тему и текст ответа
4. При необходимости настройте **расписание**:
   - Дата начала и окончания
   - Время начала и окончания

#### Пересылка писем

1. Перейдите в **Настройки** → **Расширенные**
2. Включите пересылку
3. Укажите email для пересылки
4. Выберите, сохранять ли копию в почтовом ящике

#### Алиасы email

1. В разделе **Расширенные** → **Алиасы email**
2. Введите email и имя (необязательно)
3. Нажмите **"Добавить"**

#### Язык и региональные настройки

- **Язык интерфейса**: Русский или English
- **Формат даты**: DD.MM.YYYY, MM/DD/YYYY или YYYY-MM-DD
- **Формат времени**: 24 часа или 12 часов
- **Часовой пояс**: выберите из списка

## Работа с письмами

### Поиск

Используйте поисковую строку в боковом меню для поиска писем. Поддерживаются операторы:
- `from:email@example.com` - поиск по отправителю
- `subject:тема` - поиск по теме
- `has:attachment` - письма с вложениями
- `is:unread` - непрочитанные письма
- `is:starred` - помеченные письма

### Фильтры

Быстрые фильтры доступны над списком писем:
- Непрочитанные
- С вложениями
- Помеченные
- Важные

### Папки

- **Входящие** - все входящие письма
- **Отправленные** - отправленные письма
- **Черновики** - неотправленные письма
- **Корзина** - удаленные письма
- **Спам** - спам-письма

Вы можете создавать собственные папки в настройках.

### Правила авто-сортировки

1. Перейдите в **Настройки** → **Фильтры**
2. Нажмите **"Создать правило"**
3. Настройте условия (отправитель, тема, содержимое и т.д.)
4. Укажите действия (переместить в папку, пометить, добавить метку)
5. Сохраните правило

## Продвинутые функции

### Отложенная отправка

1. При составлении письма включите **"Отложенная отправка"**
2. Укажите дату и время отправки
3. Нажмите **"Запланировать отправку"**

Письмо будет отправлено автоматически в указанное время.

### Несколько аккаунтов

1. В боковом меню нажмите на ваш email
2. Выберите **"Добавить аккаунт"**
3. Введите данные нового аккаунта
4. Для переключения между аккаунтами выберите нужный из списка

### Горячие клавиши

- `Ctrl+K` / `Cmd+K` - новое письмо
- `J` - следующее письмо
- `K` - предыдущее письмо
- `R` - ответить
- `F` - переслать
- `Delete` - удалить
- `Ctrl+/` / `Cmd+/` - показать справку по горячим клавишам

### Группировка по перепискам

Включите режим **"Переписки"** для группировки писем по темам. Это упрощает отслеживание цепочек переписки.

## Советы и рекомендации

1. **Используйте фильтры** для быстрого поиска нужных писем
2. **Настройте правила авто-сортировки** для автоматической организации почты
3. **Создайте несколько подписей** для разных ситуаций (работа, личное)
4. **Используйте отложенную отправку** для писем, которые нужно отправить в определенное время
5. **Настройте плотность отображения** под свои предпочтения

## Поддержка

При возникновении проблем:
1. Проверьте настройки подключения
2. Убедитесь, что используете правильные учетные данные
3. Очистите кеш браузера
4. Обратитесь к администратору системы
